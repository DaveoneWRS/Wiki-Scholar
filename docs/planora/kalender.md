# Kalender

Der Kalender ermöglicht die Planung von Terminen und Ereignissen, die optional mit Projekten verknüpft werden können.

## Ansicht

Der Kalender zeigt alle Ereignisse in einer Monats-, Wochen- oder Tagesansicht. Aufgaben mit Fälligkeitsdatum werden ebenfalls im Kalender angezeigt.

## Ereignis erstellen

1. Navigieren Sie zu **Kalender** in der Seitenleiste
2. Klicken Sie auf einen Tag oder Zeitslot
3. Füllen Sie die Details aus
4. Speichern Sie das Ereignis

### Eigenschaften eines Ereignisses

| Eigenschaft | Beschreibung |
|---|---|
| Titel | Bezeichnung des Termins |
| Beschreibung | Optionale Details |
| Ort | Veranstaltungsort |
| Beginn / Ende | Datum und Uhrzeit (oder ganztägig) |
| Projekt | Zuordnung zu einem Projekt |
| Farbe | Farbliche Hervorhebung |

## Ganztägige Ereignisse

Aktivieren Sie die Option **Ganztägig**, wenn kein fester Zeitrahmen benötigt wird – z. B. für Feiertage oder mehrtägige Veranstaltungen.

## Wiederkehrende Ereignisse

Termine können wiederkehrend eingestellt werden, analog zu den [wiederkehrenden Aufgaben](./aufgaben#wiederkehrende-aufgaben).

## Aufgabe aus dem Kalender erstellen

Direkt aus dem Kalender heraus können neue Aufgaben mit einem voreingestellten Fälligkeitsdatum angelegt werden – praktisch für die schnelle Tagesplanung.
