# Dashboard

Das Dashboard ist die Startseite nach der Anmeldung. Es bietet eine schnelle Übersicht über den aktuellen Arbeitstag und offene Aufgaben.

## Übersicht

Das Dashboard zeigt folgende Bereiche:

- **Heutige Aufgaben** – alle Aufgaben mit Fälligkeit heute
- **Überfällige Aufgaben** – Aufgaben deren Frist bereits abgelaufen ist
- **Zuletzt bearbeitet** – kürzlich erstellte oder geänderte Aufgaben
- **Projektzusammenfassung** – Anzahl offener Aufgaben je Projekt

## Schnellstatistiken

Oben auf dem Dashboard werden folgende Kennzahlen angezeigt:

| Kennzahl | Beschreibung |
|---|---|
| Offene Aufgaben | Gesamtanzahl aller noch nicht abgeschlossenen Aufgaben |
| Heute erledigt | Aufgaben, die heute als fertig markiert wurden |
| Überfällig | Aufgaben mit überschrittenem Fälligkeitsdatum |
| Projekte | Anzahl aktiver Projekte |

## Aufgabenangebote

Wenn Ihnen eine Aufgabe von einer anderen Person zugewiesen wurde, erscheint diese im Dashboard als **Aufgabenangebot**. Sie können die Aufgabe dort direkt **annehmen** oder **ablehnen**.

## Filter: Arbeit / Privat

Mit dem Filter oben rechts können Sie zwischen Ihren Arbeits- und Privatprojekten wechseln, um nur die relevanten Aufgaben anzuzeigen.
