# Schilder

Die Schilder-Funktion ermöglicht das Erstellen digitaler Statusschilder – z. B. als Türschild oder Kiosk-Anzeige.

## Einsatzbereiche

- **Türschild** – zeigt den aktuellen Status einer Person oder eines Raums an (z. B. „Im Gespräch", „Nicht stören")
- **Kiosk-Anzeige** – für öffentliche Displays im Eingangsbereich oder Flur

## Schild erstellen

1. Navigieren Sie zu **Schilder** in der Seitenleiste
2. Klicken Sie auf **Neues Schild**
3. Füllen Sie die Details aus
4. Aktivieren Sie das Schild

### Eigenschaften eines Schildes

| Eigenschaft | Beschreibung |
|---|---|
| Titel | Kurze Statusbezeichnung (z. B. „Besprechung") |
| Beschreibung | Optionaler Zusatztext |
| Farbe | Rot / Gelb / Grün als Statussignal |
| Ziel | Türschild, Kiosk oder beides |
| Gültig ab / bis | Optionaler Zeitraum für automatisches Aktivieren/Deaktivieren |
| Aktiv | Ein-/Ausschalten des Schildes |

## Vorlagen

Häufig verwendete Schilder können als **Vorlagen** gespeichert werden. So lassen sich Standardsituationen schnell aktivieren, ohne jedes Mal alles neu einzutragen.

## Anzeige aufrufen

Die öffentliche Schild-Anzeige ist über die **Türschild-URL** erreichbar. Diese URL wird in den Einstellungen konfiguriert und kann auf einem Tablet oder Display geöffnet werden.

:::info
Die Anzeige aktualisiert sich automatisch und benötigt keine Anmeldung.
:::
