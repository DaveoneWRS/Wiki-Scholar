# Einstellungen

Unter **Einstellungen** passen Sie Planora an Ihre persönlichen Bedürfnisse an und verwalten Zugangsdaten sowie Module.

## Persönliche Einstellungen

| Einstellung | Beschreibung |
|---|---|
| Passwort ändern | Neues Passwort festlegen |
| Desktop-Benachrichtigungen | Benachrichtigungen für Aufgaben aktivieren |
| Dunkelmodus | Helles oder dunkles Design |

## Module

Planora ist modular aufgebaut. Jedes Modul kann einzeln aktiviert oder deaktiviert werden. Nicht benötigte Bereiche werden aus der Navigation ausgeblendet.

| Modul | Beschreibung |
|---|---|
| Projekte | Aufgaben, Projekte, Kalender, Labels |
| Zeiterfassung | Arbeitszeiterfassung und Berichte |
| Schilder | Türschilder und Kiosk-Anzeigen |
| Server | Serververwaltung (Admin) |
| Statistiken | Auswertungen und Analysen (Admin) |
| Benutzer | Benutzerverwaltung (Admin) |

## API-Token

Für die Nutzung der Planora-API können persönliche **API-Token** erstellt werden:

1. Öffnen Sie **Einstellungen → API-Token**
2. Klicken Sie auf **Token erstellen**
3. Vergeben Sie einen Namen
4. Kopieren Sie den generierten Token – er wird nur einmal angezeigt

:::warning
Behandeln Sie API-Token wie Passwörter. Teilen Sie diese nicht mit anderen Personen.
:::

## Türschild-Einstellungen

Konfigurieren Sie hier die öffentliche URL Ihres Türschildes sowie das Layout der Kiosk-Anzeige.

## Administrationsbereich

Admins haben zusätzlich Zugriff auf:

- **Benutzerverwaltung** – Benutzerkonten anlegen, bearbeiten und löschen
- **Modulzuweisung** – Module pro Benutzer freischalten
- **Lizenz & Updates** – Versionsinformationen und Update-Verwaltung
- **Changelog** – Versionshistorie der Anwendung
