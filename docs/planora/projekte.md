# Projekte

Projekte helfen dabei, Aufgaben thematisch zu bündeln und den Überblick über größere Vorhaben zu behalten.

## Projekt erstellen

1. Navigieren Sie zu **Projekte** in der Seitenleiste
2. Klicken Sie auf **Neues Projekt**
3. Vergeben Sie einen Namen, optional Farbe und Icon
4. Speichern Sie das Projekt

### Eigenschaften eines Projekts

| Eigenschaft | Beschreibung |
|---|---|
| Name | Bezeichnung des Projekts |
| Beschreibung | Optionale Kurzbeschreibung |
| Farbe | Farbige Markierung zur schnellen Wiedererkennung |
| Icon | Visuelles Symbol für das Projekt |
| Typ | Arbeit oder Privat |

## Projektstruktur (Unterprojekte)

Projekte können hierarchisch verschachtelt werden. Ein Projekt kann beliebig viele **Unterprojekte** enthalten. So lassen sich komplexe Vorhaben übersichtlich gliedern.

Beispiel:
```
Schuljahr 2025/26
├── Klassenfahrten
├── Prüfungen
│   ├── Abitur
│   └── Klassenarbeiten
└── Veranstaltungen
```

## Projekt freigeben

Projekte können mit anderen Benutzern geteilt werden:

1. Öffnen Sie das Projekt
2. Klicken Sie auf **Freigeben**
3. Wählen Sie den Benutzer und die Berechtigungsstufe
4. Bestätigen Sie die Freigabe

Freigegebene Projekte erscheinen beim Empfänger in seiner Projektliste.

## Projekt archivieren

Abgeschlossene Projekte können **archiviert** werden. Archivierte Projekte werden in der normalen Ansicht ausgeblendet, bleiben aber mit allen Aufgaben erhalten und können jederzeit wiederhergestellt werden.

## Aufgaben im Projekt

In der Projektansicht sehen Sie alle zugehörigen Aufgaben inklusive ihrer Anzahl offener Punkte. Von hier aus können Sie direkt neue Aufgaben im Kontext des Projekts erstellen.
