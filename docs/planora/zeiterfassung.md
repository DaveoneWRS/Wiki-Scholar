# Zeiterfassung

Die Zeiterfassung ermöglicht eine detaillierte Protokollierung von Arbeitszeiten, Urlaub, Krankheit und Überstunden.

## Arbeitszeit erfassen

### Manueller Eintrag

1. Navigieren Sie zu **Zeiterfassung**
2. Klicken Sie auf **Neuer Eintrag**
3. Tragen Sie Start- und Endzeit ein
4. Pausenzeiten werden automatisch abgezogen (konfigurierbar)
5. Speichern Sie den Eintrag

### Stempeluhr (Ein-/Ausstempeln)

Alternativ können Sie die Stempeluhr nutzen:

- **Einstempeln** – startet die laufende Zeiterfassung
- **Ausstempeln** – beendet die Erfassung und erstellt automatisch einen Eintrag

## Eintragstypen

| Typ | Beschreibung |
|---|---|
| Arbeit | Reguläre Arbeitszeit |
| Krank | Krankheitstag |
| Urlaub | Urlaubstag |
| Feiertag | Gesetzlicher Feiertag |
| Freizeitausgleich | Abbau von Überstunden |
| Sonstiges | Sonstige Abwesenheit |

## Überstunden

Überstunden werden automatisch berechnet, wenn die erfasste Arbeitszeit die Sollarbeitszeit übersteigt. Eine **Begründung** für Überstunden kann hinterlegt werden.

## Zeitsaldo

Das **Zeitsaldo** zeigt die Differenz zwischen geleisteter und Sollarbeitszeit:

- **Positives Saldo** – Überstunden vorhanden
- **Negatives Saldo** – Minusstunden vorhanden

Der Saldo berücksichtigt:
- Erfasste Arbeitszeiten
- Urlaubstage und Krankheitstage
- Feiertage
- Einen einstellbaren Anfangssaldo

## Arbeitszeitmodell

Unter **Zeiterfassung → Einstellungen** konfigurieren Sie Ihr persönliches Arbeitszeitmodell:

| Einstellung | Beschreibung |
|---|---|
| Sollstunden je Wochentag | Ziel-Arbeitszeit pro Tag (Mo–So) |
| Urlaubstage pro Jahr | Anspruch an Urlaubstagen |
| Übertrag | Resturlaub aus dem Vorjahr |
| Anfangssaldo | Startwert für die Stundenbilanz |
| Gültigkeitszeitraum | Ab wann gilt dieses Modell |

## Berichte

Unter **Zeiterfassung → Berichte** können Auswertungen nach Zeitraum und Projekt erstellt werden.

## Urlaubs- und Abwesenheitskalender

Eine visuelle Übersicht aller Urlaubs-, Kranken- und Feiertage ist im Kalender der Zeiterfassung integriert. Halbe Tage werden ebenfalls unterstützt.
