# Fallmanagement & Kiosk

Das Fallmanagement-Modul ermöglicht die digitale Abwicklung von Formularen und Vorgängen – sowohl durch Mitarbeitende als auch über einen öffentlichen Kiosk-Touchscreen.

## Fälle verwalten (Admin)

Administratoren können Fälle (Formulare/Vorlagen) erstellen und verwalten:

- **Fall erstellen** – neuen Vorgang anlegen
- **Fall bearbeiten** – Informationen aktualisieren
- **Fall löschen** – Vorgang entfernen
- **Sichtbarkeit** – Fall aktivieren oder deaktivieren

## Vorlagen

Fälle können aus verschiedenen Vorlagentypen erstellt werden:

| Typ | Beschreibung |
|---|---|
| Festes Format | Strukturiertes Formular mit vorgegebenen Feldern |
| Einfache Vorlage | Flexibles Freitextformular |
| PDF-Vorlage | Formular basierend auf einer PDF-Datei |

**Aktionen:**
- Vorlage erstellen, bearbeiten und löschen
- Vorlage herunterladen

## Einreichungen

Eingereichte Formulare können eingesehen und verwaltet werden:

| Aktion | Beschreibung |
|---|---|
| Einreichungen anzeigen | Alle Einreichungen eines Falls |
| Details anzeigen | Vollständige Formulardaten einsehen |
| Unterschriebenes PDF herunterladen | Ausgefülltes und signiertes Dokument |
| Formular-PDF herunterladen | Blanko-Formular herunterladen |
| Einreichung löschen | Vorgang aus dem System entfernen |

## Kiosk-Interface

Das Kiosk-Interface ist ein **öffentliches, passwortfreies** Touch-Interface für Tabletts oder Displays im Eingangsbereich:

### Funktionen für Nutzende
- Verfügbare Fälle anzeigen
- Formular ausfüllen und einreichen
- Identifikation per Barcode-Scan
- Erfolgsmeldung nach Einreichung

### Kiosk-Meldungen
Administratoren können Meldungen für das Kiosk-Display erstellen:

- **Meldung erstellen** – Text und Priorität festlegen
- **Meldung bearbeiten** – Inhalt aktualisieren
- **Meldung löschen** – Meldung entfernen

Nutzende können Meldungen auf dem Kiosk lesen und als gelesen markieren. Meldungen können optional passwortgeschützt sein.

### Technische Hinweise
- Das Kiosk-Interface aktualisiert sich automatisch
- Keine Anmeldung erforderlich
- Optimiert für Touch-Bedienung
