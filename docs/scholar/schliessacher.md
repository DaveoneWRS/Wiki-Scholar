# Schließfächer

Das Schließfachmodul verwaltet alle Schließfächer der Schule – von der Standortverwaltung bis zur Zuweisung an Schülerinnen und Schüler.

## Übersicht

Das Schließfach-Dashboard zeigt:
- Gesamtanzahl der Fächer
- Belegte und freie Fächer
- Fächer in Wartung

## Standorte

Schließfächer sind nach Standorten organisiert (z. B. Flure, Gebäude):

- Standort erstellen, bearbeiten und löschen
- Standort aktivieren oder deaktivieren

## Schließfächer verwalten

| Aktion | Beschreibung |
|---|---|
| Fach erstellen | Neues Schließfach anlegen |
| Fach bearbeiten | Informationen aktualisieren |
| Fach sperren / entsperren | Fach temporär außer Betrieb nehmen |
| Wartungsstatus | Fach als in Wartung markieren |
| Schloss zuweisen | Kombinationsschloss einem Fach zuordnen |
| Schloss entfernen | Schloss von einem Fach lösen |

## Kombinationsschlösser

Kombinationsschlösser werden separat verwaltet und den Fächern zugewiesen:

| Aktion | Beschreibung |
|---|---|
| Schloss erstellen | Neues Schloss mit Code erfassen |
| Schloss bearbeiten | Code oder Status ändern |
| Schloss sperren | Schloss deaktivieren |
| Als verloren melden | Verlorenes Schloss markieren |
| PIN einsehen | Code anzeigen (eingeschränkte Berechtigung) |

## Zuweisungen

### Schüler einem Fach zuweisen
1. Schließfach auswählen
2. Schüler suchen und zuordnen
3. Pfand und Miete als bezahlt markieren

### Rückgabe
1. Schüler und Fach auswählen
2. Rückgabe bestätigen

### Weitere Aktionen
- Ersatzschlüssel hinzufügen
- Zuweisungshistorie einsehen
- Aktuellen Nutzer eines Fachs suchen

## Mieteinstellungen

In der Konfiguration können Mietgebühren, Pfandbeträge und weitere Parameter für die Schließfachvermietung festgelegt werden.
