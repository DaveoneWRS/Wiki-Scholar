# Bibliothek

Das Bibliotheksmodul verwaltet den gesamten Medienbestand der Schulbibliothek – von der Katalogisierung bis zur Ausleihverwaltung.

## Medienbestand

### Bücher verwalten
- Bücher suchen und anzeigen
- Neues Buch erfassen
- Buchinformationen bearbeiten
- Buchcover abrufen
- Klassifikation zuweisen
- Buch löschen

### Buchexemplare
Jedes Buch kann mehrere physische Exemplare haben:

- Exemplare hinzufügen und bearbeiten
- Exemplar-Barcode drucken
- Beschädigte oder verlorene Exemplare markieren

## Katalogpflege

Folgende Metadaten können verwaltet werden:

| Bereich | Beschreibung |
|---|---|
| Verlage | Verlagsnamen und -daten |
| Medientypen | z. B. Buch, DVD, Zeitschrift |
| Sprachen | Sprachzuordnung für Medien |
| Schlagworte | Inhaltliche Schlagworte |
| Interessengruppen | Alters- oder Zielgruppen |
| Buchreihen | Zusammengehörige Serien |
| Klassifikationen | Bibliotheksklassifikation |
| Autoren / Personen | Autoren- und Herausgeberdaten |

## Ausleihe

### Ausleihe durchführen
1. Leserausweis / Barcode des Ausleihers scannen
2. Buch-Barcode scannen
3. Ausleihe bestätigen

### Rückgabe
1. Buch-Barcode scannen
2. Rückgabe bestätigen

### Verlängerung
Ausgeliehene Bücher können verlängert werden:
- Verlängerungsdauer wird durch die Bibliothekskonfiguration vorgegeben

### Ausleihübersichten
- Alle aktiven Ausleihen anzeigen
- Ausleihhistorie eines Lesers einsehen
- Ausleihhistorie eines Buchs einsehen
- Überfällige Ausleihen anzeigen

## Mahnungen

Bei überfälligen Büchern können Mahnungen versendet werden:

- Mahnung für einzelne Ausleihe senden
- Alle überfälligen Leser auf einmal benachrichtigen

## Reservierungen

Bücher, die gerade ausgeliehen sind, können reserviert werden:

| Aktion | Beschreibung |
|---|---|
| Reservierung erstellen | Vormerken, wenn Buch verfügbar |
| Reservierung stornieren | Reservierung aufheben |
| Reservierung erfüllen | Reserviertes Buch ausgeben |
| Erinnerung senden | Leser über verfügbares Buch benachrichtigen |

## Konfiguration

In den Bibliothekseinstellungen können folgende Parameter angepasst werden:

- Ausleihfrist (Standarddauer)
- Mahngebühren
- Weitere bibliotheksspezifische Einstellungen

## Statistiken

Die Bibliotheksstatistik bietet Auswertungen zu:
- Ausleihhäufigkeiten
- Meistgelesene Bücher
- Aktivste Leserinnen und Leser
